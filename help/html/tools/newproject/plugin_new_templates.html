HQ Project Templates

Hypclipse contains various template wizards which can be used to create content during HQ Plug-in project creation.

Simple Server

This wizard creates a plug-in with one autodiscovered server using buildin discovery method.

Custom Autodiscover Server

This wizard creates a plug-in with one auto-discovered server using custom detector. This wizard is meant to discover server which consists with multiple server processes identified by unique process name. Every server process is attached to HQ model structure as separate service.

Win Performance Counters

This wizard creates a plug-in with one server which contains few examples how to collect metrics from Windows performance counters.

Custom Platform Service

This wizard creates a plug-in with custom auto-discovery for platform service. Custom collector is also created.
