HQDE Overview

This eclipse plug-in is called hypclipse – HQDE (HQ Development Environment). Purpose of this plug-in is to let developer to concentrate to HQ plug-in development and help to learn how plug-ins are made. For experienced users this would greatly speed up development work.

HQDE tools include:

  • Forms-Based HQ Plug-In Editor: Multi-page editor that centrally manage HQ plug-in structure.
  • New HQ Plug-In Project Creation Wizards: Create a new HQ plug-in.
  • Export Wizard: Wizard that build, package and exports HQ plug-ins.
