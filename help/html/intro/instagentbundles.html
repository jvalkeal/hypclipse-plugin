Install HQ Agent Bundles

Developing custom plug-ins usually means that there is references to existing packages available from agent bundles. Hypclipse needs to know where these agent bundle are located on the machine where hypclise is used.

Go to hyperic web site www.hyperic.com and download installation package. There is no need to run HQ agent, so only extraction of the agent package is needed. Hypclipse preferences page contain list of stored agent environments, which are later used when creating new HQ Plug-in Projects.
